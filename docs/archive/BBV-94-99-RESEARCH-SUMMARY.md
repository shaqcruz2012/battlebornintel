## JSON Edge Data Structure

Each edge object contains:
```json
{
  "source": "node_id_from",
  "target": "node_id_to",
  "rel": "relationship_type",
  "note": "detailed_description",
  "amount": "$X.XM",              // Optional: investment/grant amount
  "investmentType": "Seed/Series", // Optional: funding round type
  "y": 2024                        // Year of relationship
}
```

This structure enables:
- Network graph visualization
- Timeline-based filtering
- Investor tracking
- Partnership mapping
- Funding flow analysis
