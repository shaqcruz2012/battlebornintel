## Strategic Insights

### 1. Enterprise SaaS Dominance
Four of six companies (100, 101, 102, 105) are enterprise B2B SaaS. This aligns with Battle Born Venture's stated thesis of enterprise software. Average Series A: $6.95M.

### 2. Data Infrastructure Trend
Three companies (100, 102, 105) in data/finance operations space. Boldstart Ventures 2x portfolio concentration here, suggesting venture thesis validation.

### 3. Healthcare AI Premium
AuroraAI commands 32% larger Series A ($9.5M vs. $7.2M average) due to:
- Regulatory approval pathway (FDA 510(k))
- Higher clinical validation costs
- Longer sales cycles to hospital systems

### 4. Corporate VC as Lead Investor
Rare but emerging:
- Stripe led AuthentiPay seed round
- Google/Snowflake strategic co-investors (not leads)
Strategic investors typically co-lead or follow, rarely lead rounds.

### 5. University Partnerships are Specialized
Not all companies have university partnerships. Strongest in:
- Healthcare (AuroraAI: 3 medical schools)
- Data science (AuraData Systems: MIT CSAIL, CMU)
- Cryptography/Finance (AuthentiPay: Carnegie Mellon)

### 6. Government Grants Target Deep-Tech
SBIR programs active for companies with significant IP/R&D:
- 50% of portfolio companies pursuing SBIR
- Grants total $3.65M (6% of equity funding)
- Program progression: Phase I → Phase II (18-24 months)

## Integration Guide

### Entity Prefix Conventions

```
i_sequoia        = VC Firm (investor prefix i_)
x_stripe_v       = Corporate VC (external prefix x_)
x_databricks     = Corporate Partner (external prefix x_)
u_stanford_cs    = University (university prefix u_)
v_sbir           = Government Program (vendor/program prefix v_)
f_bbv            = Fund (fund prefix f_)
```

### Recommended Edge Fields

Essential fields for all edges:
```javascript
{
  source: "entity_id",      // Source entity
  target: "c_100-105",      // Company target
  rel: "relationship_type", // invested_in, partners_with, funded_by, etc.
  note: "description",      // Detailed relationship context
  y: 2024,                  // Year of relationship
}
```

Extended fields for investment edges:
```javascript
{
  investment: 1000000,      // Amount in USD
  stage: "Series A",        // Funding stage
  lead: true                // Lead investor indicator
}
```

Extended fields for grant/program edges:
```javascript
{
  investment: 150000,       // Grant amount
  y: 2023,                  // Award year
  program: "SBIR Phase I"    // Program name
}
```
