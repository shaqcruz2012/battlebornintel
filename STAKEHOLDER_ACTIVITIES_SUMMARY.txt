================================================================================
API ENDPOINTS
================================================================================

GET /api/stakeholder-activities
Query params: location, since, until, limit (1-200), type
Returns: Array of activities with metadata

GET /api/stakeholder-activities/company/:companyId
Query params: limit (default 20)
Returns: Activities for specific company

GET /api/stakeholder-activities/location/:location
Query params: since, until
Returns: Activities filtered by location and date range

GET /api/stakeholder-activities/stats/by-type
Returns: Count of activities by type

GET /api/stakeholder-activities/stats/by-location
Returns: Count of activities by location
